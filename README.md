#Description
a `hiredis` library wrapper for Using Redis in MT4. 


#Usage & Compile
1.Modify and Compile source code to a dll library (use Visual Studio 2010)

2.Import dll to MT4, Apply test EA to graph.

3.Test Redis connection if is work
